# My Pygame Project

This repository contains a simple Python project with a guessing game and a text-based adventure game.

## Files:
- `guessing_game.py`: A Python script where the computer guesses a number.
- `main.py`: A simple text-based adventure game.
- `build.sh`: A shell script to run the main game and install dependencies.
- `.github/workflows/build.yml`: GitHub Actions workflow to automate building and running the Python scripts.
- `.gitignore`: Specifies intentionally untracked files to ignore.

## How to Run Locally:
1.  Ensure you have Python 3 installed.
2.  Navigate to the project directory.
3.  Run the `build.sh` script:
    ```bash
    ./build.sh
    ```

## GitHub Actions:
This project uses GitHub Actions to automatically run the Python scripts on every push to the repository.
